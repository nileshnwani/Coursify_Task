Recruitment Startup Project
This project is a recruitment startup web application that allows companies to hire software engineers, product managers, data scientists, and other tech professionals. The application provides a platform where leading Indian technology companies can compete to hire top talent.

Features
Loved by leading staffing teams around the world.
Find your next top tech job in 1 week.
Companies can post job listings with titles, descriptions, locations, salaries, and qualifications.
Job seekers can apply for jobs by submitting their cover letters.
Custom user model to handle company representatives and job seekers. 
Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap, React JS
Backend: Django / Python
Database: PostgreSQL
How to Use
Clone the repository to your local machine.
Set up a virtual environment and install the required dependencies using pip install -r requirements.txt.
Create a PostgreSQL database and update the database settings in the settings.py file.
Apply the database migrations using python manage.py migrate.
Run the development server using python manage.py runserver.
Visit http://127.0.0.1:8000/ in your web browser to access the home page.
